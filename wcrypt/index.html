W3C Web Cryptography APIの遊び場

W3C Web Cryptography APIを使って作ってみたサンプルの置き場です。API自体、強い環境依存があり、動いたり、動かなかったりしますので、どうか、自己責任、ノークレームでお願いします。

RSA鍵ペアの生成
RSA鍵ペアを生成し、PKCS#8フォーマットで秘密鍵、公開鍵を表示します。
ECC鍵ペアの生成
ECC鍵ペアを生成し、PKCS#8フォーマットで秘密鍵、公開鍵を表示します。
ハッシュ計算
UTF-8のテキストに対してSHA-1/256/384/512でハッシュ計算した結果を表示します。
RSA署名と検証
UTF-8のテキストに対してRSA署名と検証が行えます。 鍵はPKCS#8 PEM形式やJWK形式でインポートしたものも使えますし、 ブラウザで鍵ペア生成したものも使えます。
